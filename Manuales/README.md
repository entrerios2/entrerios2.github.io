# Manuales de Dispositivos

Esta carpeta es el repositorio central para todos los manuales de fabricantes, hojas de especificaciones técnicas y guías de operación para el equipamiento utilizado en nuestra instalación audiovisual.

## Documentos Esperados
- Consola digital Behringer XR12
- Micrófonos (Sistemas inalámbricos y de atril)
- Amplificadores de potencia y parlantes (Wharfedale, RCF, DB Technologies)
- Manuales de routers y redes
- Cualquier otra tecnología de video / proyección (Ej. Blackmagic ATEM)

*¡Por favor, coloque toda la documentación en formato `.pdf` aquí!*
