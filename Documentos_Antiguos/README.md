# Documentos Antiguos / Archivo

Por favor, guarde sus archivos de inventario exportados (Google Sheets en `.csv` o `.xlsx`), PDFs antiguos, archivos de texto o cualquier información de referencia heredada dentro de esta carpeta.

Una vez que estén aquí, podré leerlos para extraer y consolidar todos los datos de conexiones y stock para organizar nuestra nueva documentación y configuración.
