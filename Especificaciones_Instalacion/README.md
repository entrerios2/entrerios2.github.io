# Especificaciones de Instalación

Esta carpeta contiene las dimensiones físicas, el diseño arquitectónico y los detalles de distribución del hardware para el recinto del evento.

## Documentos y Datos Planeados
- **Dimensiones del Salón**: Medidas físicas del espacio, altura del techo, materiales (para consideraciones acústicas).
- **Distribución de Parlantes**: Ubicación física del PA Principal (Front), Líneas de Delay, Monitores y rellenos de sala auxiliar.
- **Ubicación de Micrófonos**: Distribución en atril, escenario y micrófonos interactivos para la audiencia.

*¡Una vez que se definan las dimensiones y distancias necesarias (como las que necesitamos para configurar las líneas de retardo de la XR12), las documentaremos aquí!*
